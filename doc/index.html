Anyone interested in testing this should first reset the database with

rails db:migrate:reset

then populate the database with

rails:db:seed

–

All user accounts must be confirmed via email. Check the test.rb environment if you wish to use a tool to catch emails. I suggest mailcatcher.

There is only one admin account and you cannot create another one via the web app. The credentials for the admin account are in seeds.rb
